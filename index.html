Methodology

  • Scope: primarily 100B+ public models and closed frontier models whose exact parameter counts are not disclosed but are clearly positioned as flagship large LLMs.
  • Intelligence index: generated from a curated benchmark-consensus seed and blended with machine-readable benchmark signals when adapters expose stable scores.
  • Cost: OpenRouter API pricing converted to dollars per million tokens. Blended cost = input $/M + output $/M.
  • Value index: intelligence score divided by blended cost, then normalized for readability. Very low-cost models can rank high only if their intelligence score remains competitive.
  • Warning: benchmark rankings move fast. Treat this as a website foundation with visible source links, not a permanent truth table.
